在學經歷

您好，我是楊于曜，畢業於亞洲大學資訊工程學系。在學期間，我主要使用 Python 和 C# 語言開發，具備Django、Unity專案經驗。

善於以抽象化思維分析需求並解決實際問題，且樂於與他人討論與學習新技術。

為了補足理論知識，我經常透過網路資源自學，並透過實作專案來累積開發經驗，包括網路爬蟲、遊戲開發與網站建置等。

校內專案/研究

  • 參與校內 NLP 相關研究計畫的，利用爬蟲技術收集百萬筆資料，為計畫貢獻文本處理資料集。

  • 利用socket技術整合骨架辨識和Unity，達成復健遊戲的虛實整合。

  • 將Django購物網站建立於aws ec2上，並設定public ip，讓使用者在外網也可瀏覽網站

  • 使用.net core開發網站，EFcore實作基本CRUD，並加入authentication隔離各使用者管理的資料

學程與證照

在大四期間，我參與校內 AWS 系統架構與應用服務學程，熟悉AWS基礎服務，並取得 AWS Certified Cloud Practitioner 證照。在學程中，我學習了無伺服器架構及各種架構設計的應用場景，同時透過團隊合作，深刻體會如何在團隊中發揮自身價值。

實習經驗

此外，在企業實習期間，參與業界專案，實際接觸 MVC 架構、RESTful API 設計，並使用 FastAPI 與 Docker 進行開發。除了技術上的成長，我也累積了與主管、同事溝通協調的經驗，學會如何在職場中清晰且精確地表達想法，確保團隊目標的一致性。

未來展望

未來，我希望更加提升後端開發能力，參與更多高效能與可擴展性的系統開發，並在團隊合作中不斷學習與成長。

期待能夠在貴公司進一步挑戰大型專案，並持續提升後端技術能力
