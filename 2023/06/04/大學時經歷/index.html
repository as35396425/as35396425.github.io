以下自介

您好，我是楊于曜，畢業於亞洲大學資訊工程學系。在學期間，我主要使用 Python 和 C# 語言開發，具備Django、Unity專案經驗，以下是我大學時的經歷。

hexo+S3建構個人部落格

部落格包含標籤雲以及彙整功能，便於記錄個人的學習軌跡，目前的頁面也是建構在個人的部落格當中。

建構此部落格需要在AWS中開啟S3的靜態網站託管功能，以及配置bucket Policy 允許所有人獲取物件。

Hexo deploy還需要配置aws CLI 的credentials，可以透過系統變數設定，或是使用awsSDK的configure配置權限。

冷氣規格與型號價格查詢

這是在就業學程當中老師帶我們做的專題，專題架構是採用無伺服器架構，使用S3架設了靜態網站，在網站中呼叫在API Gateway建立好的API，之後根據所呼叫的API觸發不同的Lambda，並配合dynamoDB進行，當中遇到比較困難的點是需要設定映射範本，讓使用者所查詢的資料能包成json檔，並傳到lambda查詢輸入內容，之後才發現可以使用”$input.json來接收使用者輸入

車牌辨識 Rekognition +API gateway +Lambda +LineNotify

利用上傳圖片檔到S3作為Trigger呼叫Lambda，在Lambda中寫好呼叫Rekognition程式，並利用API Gateway接收使用者上傳圖檔，再經由Lambda上傳到S3

Unity虛實整合復健遊戲

簡介

利用AzureKinect抓取使用者復健動作，追蹤全身32個骨架節點，以此繪製骨架或操作遊戲人物。

利用遊戲形式增添趣味性改善療程的枯燥乏味，且將傳統復健場所遷移到虛擬遊戲，降低病人通勤的時間成本。
復健動作包括： 起身坐下、原地踏步…等，根據復健動作的準確度，動作越標準得分數越高，起身的時間間隔與次數也會納入計分考量。
